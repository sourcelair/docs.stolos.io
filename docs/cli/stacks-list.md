List your stacks.

## Usage

```
Usage: stolos stacks list [OPTIONS]

  List your stacks

Options:
  --stolos-url TEXT  The URL of the Stolos server to use, if not the default
  --help             Show this message and exit.
```

## Example

```
$ stolos stacks list
Stack name    Slug
------------  --------------------
mystack       mycompany/mystack
otherstack    mycompany/otherstack
```
