Open the public URL of the current project.

## Usage

```
Usage: stolos launch [OPTIONS]

  Open the public URL of the current project

Options:
  --help  Show this message and exit.
```

## Example
```
$ stolos launch
Opening http://sourcelair-stolos-akalipetis-zzqouw.sourcelair.stolos.io...`
```
